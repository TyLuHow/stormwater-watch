# Contributing to Stormwater Watch

Thank you for your interest in contributing to environmental protection!

## Code of Conduct

This project serves the public interest. We expect all contributors to:
- Prioritize environmental protection and community health
- Respect the nonprofit mission
- Maintain data accuracy and provenance
- Follow responsible disclosure for any security issues

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Ensure all data sources remain public and properly attributed
5. Submit a pull request

## Development Setup

See README.md for setup instructions.

## Reporting Issues

- Use GitHub Issues for bug reports and feature requests
- For security issues, please email directly (don't post publicly)

## Data Integrity

- Never include private or authenticated data sources
- Maintain clear provenance for all data
- Validate all calculations affecting violation detection
