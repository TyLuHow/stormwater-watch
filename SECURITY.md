# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

**DO NOT** create public issues for security vulnerabilities.

Instead, please email: [security contact needed]

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Security Considerations

This platform handles:
- Public environmental data only
- No personal health information
- No payment processing
- Email addresses for alerts only

All legal notices generated require attorney review before use.
